Es ka matlab :- Ek naksha jo ki similar type ke object bnata hai baar baar & ek machine jo ki similar type ke product banati h.

Example :

Class ko factory smjo aur object create kro. jab bhi object create hoga toh similar hi create hoga

Lets think in english

In the real world, you'll often find many individual objects all of the same kind. There may be thousands of other bicycles in existence, all of the same make and model. Each bicycle was built from the same set of blueprints and therefore contains the same components. In object-oriented terms, we say that your bicycle is an instance (example) of the class of objects known as bicycles. A class is the blueprint from which individual objects are created.
